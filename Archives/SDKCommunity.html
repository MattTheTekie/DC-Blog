CSS Profile/Group Template | 11/15/2021 11:25 PM I have some CSS knowledge thanks to blueturtle teaching me. I have a basic template for you on your profile/group(s). Ok to start here is the template:

*{background:black!important;border-color:gold!important;color:blue!important;}
IMG[src="/images/logo.png"]{content:url(http://3dspaint.com/images/upload_gallery/15245.jpg)!important;background-attachment:fixed!important;background-repeat:none!important;background-position:center!important;height:100px;width:300px;} Body{background:url(http://3dspaint.com/images/upload_gallery/15261.jpg)!important;background-attachment:fixed!important;} :hover{color:crimson!important;}

If you don't care what it looks like, just copy and paste it into the CSS box. To go to that you first click your profile picture. After you are in My Zone - Maintenance, you click Settings. Go down to Profile Styling (CSS). After you copy and paste this.

Odds are you want to change it though. To do so is very easy. For the top part, you can change it from 3DSPaint to any image. First go to 3dspaint.com (not dsipaint.com or sdkcommunity.com), then you click on your profile.

After that you click Image Uploads in My Zone - Maintenance.
Then you click Choose File, choose a photo, name it, then check the hard to see box that says that you agree all images are yours and you have the right to upload them.

Then you copy the link to said photo (i.e. http://3dspaint.com/images/upload_gallery/15214.jpg). Do that by (on PC) dragging the photo to the search bar. If you can't do that you need another method.

Take the first part of the example link (http://3dspaint.com/images/upload_gallery/?????) and add the number after the "[FIGURE" in the photo uploaded success details. From there you copy the link into:

{content:url(InsertLinkHere)!important;background-attachment:fixed!important;background-repeat:none!important;background-position:center!important;height:100px;width:300px;}

You can easily change the size of it by changing "height:?px;width:?px" in the code. Height=How Tall and Width=How Long. Instead of height:100px;width:300px you can have height:50px;width:150px as an example. Pretty useful.

You don't have to use 3DSPaint uploads and can do so in another place that does something similar to Image Uploads on 3DSPaint. Backgrounds are bigger so choose a slightly bigger photo if you want. Doing the Background is very similar but in a different location. Using the same methods, you copy and paste into:

Body{background:url(LinkHerePlease)!important;background-attachment:fixed!important;}

How to change text color:

You can change text color by editing:

*{background:black!important;border-color:gold!important;color:blue!important;}

background:black is for the background color. You can change it to something like blue, green, purple, gold, etc.

border-color:gold is for changing the border of the boxes surrounding things like the Biography, Badges, and Last Blogs.

important;color:blue is the color of the text itself.

:hover{color:crimson! is for the color of the text when you hover over stuff like the text and boxes.

For every single one of them, you need to change the part where it says a color like gold with something else like crimson. That way, you will see different colors.

Note that unless the CSS is applied to the profile, group, blog, or whatever else, it won't display but it means it kinda could work outside of paint if you re-write some of the CSS.
Navigator Avatar Changing To use it, technically all you have to do is copy and paste the code below.

div.nav_avatar {content:url(New-Image-Here)!important}

"div.nav_avatar" is the element for the avatar icons. "content:url()!important" is where you put your image. Put the image inside of the () part. I would recommend making it 64x64 at most, maybe 32x32. If your image is hosted via 3DSPaint, you can just remove "http://3dspaint.com" and it will work even on DSi.

This will change it for everyone as a warning and only changes the navigation avatar in profiles, groups, etc. Also the link to your zone will be broken via this technically but thats not a big deal, just go to the home menu first and it will work.
Specific Profile Changes This is a bit more complicated but it allows changes per person. If they change their PFP, you will have to change the CSS again so unless they never change it or its for yourself in groups, I wouldn't recommend this one.

img.avatar[src="Your-Avatar-Image-Here"] {content:url(New-Image-Here)!important}

Now this will change ur PFP for the group which is very nice, but you have to find where your profile picture is stored. If you need help, I will get that link for you and put it in the CSS for you to copy/paste.

It probably works on profiles as well. I won't personally use this much because its gonna be annoying unless I can figure out how to get it to work for a ranking system in TDK, but feel free to use this for anything you want. Once again would recommend like 64x64 size at most if you want it to work right on DSi.
And thats how you do custom CSS profile pictures even after the PFP uploader broke. I might mess around with it to see if I can get a ranking system going but I'm not sure. I'll probably just use it for my own PFP tbh.
